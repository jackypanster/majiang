# 血战到底麻将规则（埋三张定缺微调版）

## 概述

血战到底是四川麻将的一种流行变体，主要流行于成都及周边地区。它以“血战模式”为核心，即一家胡牌后不结束牌局，未胡牌的玩家继续打牌，直到三家胡牌或流局结束。这种规则增加了博弈性和趣味性，强调“血战到底”的逆转可能。游戏使用筒、条、万三色牌，共108张，无花牌、无字牌。玩家必须缺一门花色（即胡牌时手牌只能有两种花色）才能胡牌。

**游戏人数**：4人。  
**一局结束**：三家胡牌或流局（牌墙摸完）。  
**坐庄规则**：  
- 庄家胡牌连庄；闲家胡牌上家坐庄；流局上局庄家继续；一炮多响时，第一个胡牌的玩家坐庄。

## 用牌与配牌

- **牌数**：筒子、条子、万子各1-9四张，共108张牌。  
- **配牌**：每人13张牌。庄家先摸牌（14张）。  
- **定缺埋牌**：开局定缺时，每位玩家从手牌中选择3张同一门的牌埋掉（面朝下放置，作为定缺该门的证明）。埋牌后，闲家手牌剩10张，庄家手牌剩11张。牌局结束后，检查埋牌是否为同一门花色；若不是同一门，则视为定缺无效（可扣罚底金×8分或视为花猪）。缺门牌不可碰/杠/胡，但非缺门牌可正常操作。缺门牌未打完，不可打出其他花色牌。埋掉的牌不进入牌墙，庄家先出牌，然后轮流摸牌打牌。

## 基本玩法

- **不能吃牌**：血战麻将禁止吃牌。  
- **碰牌**：任何玩家打出的牌与自己手上有两张相同时，可碰（明碰）。  
- **杠牌**：  
  - **明杠（直杠）**：手中有三张相同牌，对家打出第四张时杠。点杠者付2倍底分。  
    - **暗杠**：手中有四张相同牌，自摸杠。其他玩家付1倍底分。
    - **补杠（巴杠/面下杠）**：已碰三张，自摸第四张杠。其他玩家付1倍底分。  
  
- **打牌顺序**：出牌后轮流摸牌打牌（优先转牌/暂停）。杠后补一张牌。  
- **漏胡（过手胡）**：过胡后，同圈内只能胡更高番型牌，动牌后解除。  
- **最后一张牌**：可碰不可杠。

## 胡牌条件

胡牌必须缺一门（手牌仅两种花色）。胡牌牌型由**一对将**（11）和**三组面子**（123顺子、111刻子或1111杠）构成。

根据面子组成的不同，胡牌时的总张数会变化：
- **标准胡牌（无杠）**：11张牌（例如：11 + 123 + 456 + 777）
- **含一个杠**：12张牌
- **含两个杠**：13张牌
- **含三个杠**：14张牌

**核心定义**：胡牌的目标是凑成“一对将 + 三个面子”的结构，而非固定的11张牌。  

**胡牌类型**：  
- **自摸胡**：自己摸牌胡，三家付。  
- **点炮胡**：他人打牌胡，一家付。  
- **抢杠胡**：胡他人补杠牌。  
- **一炮多响**：允许一人点炮多人胡，先胡者为一胡。  

**特殊胡**：  
- **天胡**：庄家配牌即胡（5番）。  
- **地胡**：闲家首轮胡（5番）。  
- **杠上花**：杠后自摸胡（+1番）。  
- **杠上炮**：杠后打牌点炮（+1番）。  
- **海底捞月/扫底胡**：摸最后一张牌胡（+1番）。  

## 牌型与番数

番数决定底金倍数（1番=×2，2番=×4，3番=×8，4番=×16，5番=×32；多番叠乘）。基本胡0番（×1）。部分牌型可叠加（如门清+1番）。由于埋三张，牌型番数计算基于剩余手牌。

| 牌型名称     | 番数 | 描述与示例                                                                 | 叠加限制 |
|--------------|------|----------------------------------------------------------------------------|----------|
| 基本胡       | 0    | 任意两门牌，满足“一对将+三组面子”的结构。                                  | -       |
| 门清         | +1   | 未吃/碰/明杠（暗杠算）。                                                   | 可叠加  |
| 自摸         | +1   | 自摸胡。                                                                   | 可叠加  |
| 带根（每根） | +1   | 手牌有四张相同牌（每组+1）。                                               | 可叠加  |
| 对对胡       | 1-2  | 3副刻子/杠。示例：111 333 555 88。                                       | -        |
| 清一色       | 2    | 全一门花色。示例：全万子。                                                 | 可叠加  |
| 幺九         | 2    | 全1/9牌顺子/刻子，将也1/9。示例：111 999 123万。                           | 与中张禁 |
| 将对         | 3    | 对对胡中全2/5/8。                                                          | 不计对对 |
| 清对         | 3    | 清一色对对胡。                                                             | 不计清一/对对 |
| 清幺九       | 4    | 清一色幺九。                                                               | 不计清一/幺九 |

| 清金钩钓     | 4    | 清一色金钩钓（计对对胡）。                                                 | 不计清一/金钩 |

**注**：十八罗汉等高阶牌型已取消。夹心五（4/6夹5）在4人玩法中无效。埋三张后，牌型需适应减少的起手牌张数。

## 特殊规则：血战到底模式

- **血战流程**：一家胡后，未胡者继续打牌（已胡者停牌），直到三家胡或流局。一炮多响允许。  
- **报叫**：配牌后立即下叫（+2番）。  
- **刮风下雨**：杠分即时结算，流局或抢杠时退回（退税）。杠上炮分转给胡者。  

## 计分规则

- **底分**：1番=底金×2，叠乘计算（例：4番=16倍）。  
- **点炮**：点炮者付胡者番数分（一家付）。  
- **自摸**：三家各付番数分（庄家双倍）。  
- **杠分**：直杠2分（点杠者付）、暗杠/巴杠1-2分（每家付）。  
- **公式示例**（自摸）：总分 = (总番 + 自摸番 + 杠分) × (人数-1) - 点杠/暗杠扣分。  
- **庄家加成**：庄家胡牌番数+1。  
- **埋牌检查**：若埋牌非同一门，扣罚底金×8分（或视为花猪加重罚）。

## 流局处理（黄庄）

1. **查花猪（查卫生）**：手牌三门花色的玩家扣48分（或底金×48）。  
2. **查大叫**：未听牌者（花猪除外）赔听牌者最大可能番数分；退回杠分（退税）。不查自摸。  
3. **杠分退回**：流局时未听牌者退回杠分。  
4. **埋牌验证**：检查所有埋牌，若非同一门，相应玩家额外扣罚。

**注意**：本规则为标准版基础上添加的“埋三张定缺”微调变体。适用于4人游戏。如有进一步调整或本地差异，可讨论。
